### Prerequisities

> Python 3+
> Flask + SQLAlchemy.

### Installing

Extract the repo. files in a directory

File structure:

    flask-SQLAlchemy-Tables\
      app\
	    templates\
		  add.html
		  show.html
        __init__.py
        views.py
	  run.py
	  db_create.py
	  config.py
	  app.db
	  README.md

Run:

Start Flask:
> python run.py

Create sqlite DB (Optional):
> python db_create.py

### Usage


- To view list of users: http://127.0.0.1:5000/ or http://127.0.0.1:5000/show

- To add new user: http://127.0.0.1:5000/add
